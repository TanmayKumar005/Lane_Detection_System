# Lane_Violation_and_Recognition_System

To run this project follow these given steps:


### **Step 1:** Clone this repository

```
https://github.com/TanmayKumar005/Lane_Violation_Detection_System.git
```

### **Step 2:** After cloning create a python virtual environment and activate it


### **Step 3:** Install all the required dependencies

```
pip install -r requirements.txt
```

### **Step 4:** Create your app password using google two factor authentication for sending mail


### **Step 5:** Now create an .env file and save your app passwords like:

```
SENDER_EMAIL=your mail id
SENDER_PASSWORD=your generated password
```

### **Step 6:** Run the main.py file

```
python app.py
```


Finally view the application on Port Number 5000 or localhost
